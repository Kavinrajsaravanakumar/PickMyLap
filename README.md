# PickMyLap – Laptop Recommendation Platform

PickMyLap is a full-stack web application designed to help users find laptops based on their specific requirements. It provides filtering, comparison, and recommendation features using modern web technologies.

---

## Features

- Search and filter laptops by price, brand, RAM, storage, and processor  
- Compare multiple laptops side-by-side  
- Recommendation system based on user preferences  
- Chatbot interface for assisting users  
- User authentication (Login/Register)  
- Responsive and smooth user interface  

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)

### Other Tools
- Axios (API communication)
- TensorFlow.js (optional recommendation logic)
